# PortG 🧭

## O Teu Portfólio Tech Vivo

PortG é um hub de estudos e portfólio público para devs em formação. Organiza tecnologias, metas semanais e aprendizados enquanto gera uma página pública para recrutadores.

### Visão Geral
- Dashboard com progresso semanal, tecnologia ativa, streak e commits.
- Conexão com GitHub para sincronizar repositórios e evolução.
- Portfolio público atualizado automaticamente.
- Metas semanais com status e prioridade.

### Como rodar
\`\`\`bash
pnpm install
pnpm dev
