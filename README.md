# PortG 🧭
### O Teu Portfólio Tech Vivo

O **PortG** é um hub de estudos e portfólio público: organiza tecnologias, metas semanais e aprendizados enquanto constrói uma vitrine para recrutadores. A ideia é **aprender construindo** e terminar com um portfólio vivo.

---

## ✨ Experiência Sensorial
O PortG une prática e apresentação, sem separar estudo de carreira:

- **Dashboard Pessoal**: visão semanal, streak e tecnologias ativas.
- **Metas Semanais**: metas por tecnologia com taxa de conclusão.
- **Sync GitHub**: importa repositórios e commits publicados.
- **Portfólio Público**: página atualizada automaticamente.
- **Timeline de Evolução**: registra aprendizados e insights.

---

## 🛠️ Funcionalidades Principais

| Recurso | Descrição |
| :--- | :--- |
| **Metas Semanais** | Acompanhe taxa de conclusão e prioridades. |
| **Hub de Tecnologias** | Registre categoria, nível e status por stack. |
| **GitHub Sync** | Repositórios e commits ligados ao perfil. |
| **Portfólio Público** | Vitrine visual para recrutadores atualizada por commit. |
| **Insights** | Registro de aprendizados relevantes. |

---

## 🚀 Como Começar
1. Instale dependências com `pnpm install`.
2. Rode localmente com `pnpm dev`.
3. Crie conta e conecte o GitHub para começar o sync.
